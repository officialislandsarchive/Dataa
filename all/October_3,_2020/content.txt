October 3, 2020 Basic Information Day released October 3rd, 2020 Version number 22 Version Navigator Previous version Next version September 18, 2020 October 16, 2020 The October 2, 2020/October 3, 2020 update, or more commonly known as the Cows update, is an update in Islands. Although the update was planned to release on October 2nd it was delayed and actually released at 1:10 AM October 3rd PST. This update brings cows along with farm items. The update also brought a new industrial machine as well as a new ingot.

Changes [ ]

Additions [ ]

Items & Blocks [ ]

Added Gold Deposit . Added new weapon Rageblade along with the Rageblade Blueprint which is dropped from Buffalkors . Added Red Bronze Ingot Added Cheese Added Bucket that can be turned into a Milk Bucket when milking a cow.

NPC's [ ]

Added Taliyah who can be found near a barn on the Mainland .

Machines [ ]

Added Red Bronze Refinery to make Red Bronze Ingots

Animals & Farming [ ]

Added Cows Added Cheesemaker Added Wheat Trough Added Hay Bed Added Animal Well-being Kit Added Animal Bench

Modifications and bug fixes [ ]

Deposit islands spawn closer to your main island . Fixed an issue that caused island corruption Client performance improvements The Food Processor now requires Crystallized Gold in its recipe again instead of just gold . Buffed the crystallized gold drop rate to 10%
