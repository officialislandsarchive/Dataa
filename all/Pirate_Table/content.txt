Pirate Table Type of block Solid block Best tool Luminous Transparent Description A pirate's table! The pirate table is a furniture block that has a chance to be obtained from treasure chests .

Obtaining [ ]

Treasure chest [ ]

Opening a treasure chest has a 2% chance of rewarding a pirate table, having a rarity of 'legendary'.
