October 18, 2021 Basic Information Day released October 18, 2021 Version number 61 Version Navigator Previous version Next version October 9, 2021 November 13, 2021 The October 18, 2021 update, also known as the Halloween 2021 update, is a major update in Islands that added content for the Halloween 2021 event.

Changes [ ]

Additions [ ]

Added Percival the Pumpkin King Added Pumpkin Bats Added Pumpkin Boss pets Added Pumpkin Boss Tokens Added Pumpkin Hammer Added Cauldrons Added new furniture blocks: The Halloween Trophy Pumpkin Cat Pumpkin Bed Spooky Pumpkin Halloween Lantern Ghost Lantern Gravestone Candy Basket Cobweb

Modifications [ ]

Revamped the Hub to a new Halloween style New buildings Changed the appearance of some buildings Moved all buildings and villagers to new locations Increased the size of the Hub Lake Added an expansion for the Pumpkin Boss spawn area Changed Ivon's appearance Fixed a bug regarding the Industrial Merchant Teleseller Changed the treasure map's bird's-eye-view interface to match with the Halloween-themed Hub

Removed [ ]

Removed dialogue about the 'mystery' for all the villagers
