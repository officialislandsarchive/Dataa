December 10, 2021 Basic Information Day released December 10, 2021 Version number 64 Version Navigator Previous version Next version November 27, 2021 December 18, 2021 The December 10, 2021 update, also known as the trap door update, is a minor update in Islands that introduced trap doors and quality of life changes.

Changes [ ]

Additions [ ]

Added Trap Doors Added Coral Blocks for all coral variants

Modifications [ ]

Changed the texture of glass blocks The Turkey Trophy is now attainable from Kontos Overhauled item and coin notifications Improved the potion effect status UI Updated the treasure map with the new Hub Improved eating for mobile players Opened treasure chests no longer make sound Increased the industrial milker capacity from 3 to 6 Lighting now changes with music changes Improved lag from placing blocks

Bug fixes [ ]

Fixed the issue with Turkeys not dropping off berries Fixed the issue with searching inventory in foreign languages Fixed the bug where having multiple treasure maps prevented the usage of more than one Fixed the bug where the static scepter could spawn multiple arcs Fixed the industrial milker Reduced lag from totem animations, clouds, and glass blocks

Bug fixes (December 14, 2021) [ ]

Fixed resource islands occasionally spawning inside of other blocks Fixed block prediction allowing people to fly on islands when they don't have permission Reduced Fhanhorn pull and knockback
