Void Stone Block Void Stone Block Rock Type of block Solid block Best tool Luminous Transparent A void stone block is a block that can be mined from a void stone rock .

Obtaining [ ]

Mining [ ]

Void stone rocks naturally generate throughout the Void Isles . Mining a void stone rock yields 1 - 2 void stone blocks after each stage and 9 Mining XP (36 with the 4x daily bonus), resulting in a total of 2 - 4 void stone blocks and 18 Mining XP (72 with the 4x daily bonus) per rock.
