Pirate Ship Wheel Type of block Solid block Best tool Luminous Transparent Description A pirate ship's wheel! The pirate ship wheel is a furniture block that has a chance to be obtained from treasure chests . It can only be placed on walls.

Obtaining [ ]

Treasure chest [ ]

Opening a treasure chest has a 10% chance of rewarding a pirate ship wheel, having a rarity of 'uncommon'.
