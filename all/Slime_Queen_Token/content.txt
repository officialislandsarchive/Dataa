Slime Queen Token Dropped by Pink Slime Slime King Description A token obtained from the Slime Island! The Slime Queen token is an item used to summon the Slime Queen .

Obtaining [ ]

Mob drop [ ]

Pink slimes have a 50% chance to drop a Slime Queen token and the Slime King is guaranteed to drop 1 - 3 Slime Queen tokens upon defeat.

Dungeon Chest [ ]

Opening dungeon chests have a 10% of rewarding ?? - ?? Slime Queen tokens, having a rarity of 'uncommon'.

Usage [ ]

Summoning [ ]

At least 100 Slime Queen tokens can be used summon the Slime Queen .

History [ ]

July 2, 2022 Added Slime Queen tokens.

Trivia [ ]

This is the first token that can be dropped from a boss .
