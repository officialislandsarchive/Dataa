Pirate Chandelier Type of block Solid block Best tool Luminous Transparent Description A pirate's chandelier! The pirate chandelier is a furniture block that has a chance to be obtained from treasure chests . It can only be placed underneath blocks.

Obtaining [ ]

Treasure chest [ ]

Opening a treasure chest has a 10% chance of rewarding a pirate chandelier, having a rarity of 'uncommon'.
