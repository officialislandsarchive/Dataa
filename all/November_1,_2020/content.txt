November 1, 2020 Basic Information Day released November 1st, 2020 Version number 24 Version Navigator Previous version Next version October 16, 2020 November 7, 2020 The November 1, 2020 update, also known as the New Flowers and Colors update, introduced more colors to the coloring station , revamped flowers , and reduced lag.

Changes [ ]

Additions [ ]

Items & Blocks [ ]

More clay variations, now 11 colors in total. Added 4 new types of flowers with color variants, over 20 variations in total.

Hub [ ]

Hub revamped and counted as a separate server, instead of being part of a server. The Hub can now hold 50 people.

Chat [ ]

Added a cross-server chat system. Players are able to chat with others, despite not being in the same server.

Modifications [ ]

Islands [ ]

Every island is now counted as a server itself. This reduces lag and allows developers to increase the block limit in the future. All players now have free access to join codes.

Building [ ]

Added double doors Added upside down stairs Added top and bottom slabs Added ability to press R to rotate while placing

Other [ ]

Hotbar layout saves on rejoin. Music toggle saves on rejoin. Doubled the lifetime and amount limit of factory-dropped items (this means items produced by machines last longer when dropped) All items that previously displayed as a block when hold now display the item properly (this means you can see some type of item now) Removed Halloween content. Watering cans now have infinite water storage. Can no longer access player list. Major duplication glitch patched.
