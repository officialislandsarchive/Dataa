November 20, 2020 Basic Information Day released November 20th, 2020 Version number 26 Version Navigator Previous version Next version November 7, 2020 November 26, 2020 The November 20, 2020 update, also known as the PvP update, introduced PvP, three new villagers along with their respective currencies, and more.

Changes [ ]

Additions [ ]

PvP Alpha [ ]

Began PvP Alpha event. Added PvP Island . Added PVP Alpha Trophy which can be purchased from the Mercenary . (Limited time) Added Mercenary villager along with quests.

Flowers [ ]

Added Fulco along with quests. Added Daffodil flower which can be unlocked through Fulco. Added Pottery Bench to craft flower pots which can be purchased through the Florist.

Desert [ ]

Added Cactus crop. Added Cactus Spike sword. Added a city with nomads on Desert Island . Added quests to Azur . Added Desert Chest . Added Desert Furnace .

Currencies [ ]

Added Sand Dollars which can be obtained from Azur . Added Petals which can be obtained from Fulco . Added Skulls which can be obtained from the Mercenary .

Modifications [ ]

Vending Machines [ ]

Vending Machines now have a 10% tax on all transactions. Increased vending machine max stack size to 300. Increased vending machine max transaction price to $1B. Added commas to vending machine display prices. Added direct input for vending machine price. Added "2x" deposit button to help for large prices.

Other [ ]

Revamped Quest system. Crops now grow while the player is offline. The Hub will no longer be too bright during certain times of the day. Skorp animations now work correctly and are improved. Improved general island loading time for slower devices. Removed "Go Home" button in hub on phones.
