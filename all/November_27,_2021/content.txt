November 27, 2021 Basic Information Day released November 27, 2021 Version number 63 Version Navigator Previous version Next version November 13, 2021 December 10, 2021 The November 27, 2021 update, also known as the maple isles update, is an update in Islands that revamped the Maple Isles and added new maple-themed items and blocks .

Changes [ ]

Additions [ ]

Re-added and revamped the Maple Isles (formerly Turkey Island ) Added four new villagers : Emma Bailey Don Kontos Added Turkeys Added Raspberries Added Raspberry Seeds Added a new boss, Fhanhorn Fhanhorn's summon cooldown is now cannot be bypassed with tokens due to no tokens Added Maple Workbench Added Turkey Spawn Egg Added Fhanhorn Pet Spawn Egg Added Berry Basket Added Industrial Berry Basket Added Tree Tap Added Maple Sap Added Maple Syrup Added Maple Syrup Bottle Added Sap Boiler Added Syrup Bottler Added Pancake Dough Added Pancakes Berry Pancakes Sweet Pancakes Fhanhorn's Pancakes Added Wood Tube Added Wood Tube Merger Added Maple Shield Added Antler Shield Added Antler Hammer Added Turkey Trophy Added Fhanhorn's Flower Added new furniture blocks: Cornucopia Bed Turkey Couch Maple Couch Vanilla Turkey Bean Bag Maple Turkey Bean Bag Chocolate Turkey Bean Bag

Modifications [ ]

Slightly nerfed cooking yield from skill level (maximum of +1.3 to +1) Increased the spawn rate of buffalkors on Buffalkor Island Removed quests for Cletus and Taliyah

November 28, 2021 [ ]

Bug fixes [ ]

Fixed an issue with unlocking the recipe for Fhanhorn's pancakes. Fixed raspberry bushes getting destroyed when using sickle .

November 29, 2021 [ ]

Bug fixes [ ]

Fixed a bug which would cause items to be stuck on conveyors . Fixed the industrial berry basket. Fixed black particles on many blocks and tools. Fixed Underworld spell particles not disappearing. Fixed dialogue popup with villagers . Fixed random seats around the Hub . Fixed bugged treasure spots on treasure maps . Reduced lag in the Hub.
